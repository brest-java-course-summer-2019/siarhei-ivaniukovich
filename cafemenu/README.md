## Cafe project
 Cafemenu - application for cafe employees to control of making orders (adding/editing items), send them to job queue, check and control process, and closing after pay confirmed.

## Prerequisites
- jdk11  
- maven 3+  

  
## Installing  
`mvn clean install`  
  
## Running the tests  
`mvn clean test`
